# Dota 2 Advanced Mechanics Assistant

An open-source educational tool designed to help players deepen their understanding of Dota 2's complex mechanics. This project is a **training aid and reference guide**, not a modification or hack for the live game client.

## Features (Educational & Informational)

*   **Last Hit Training Module:** A standalone simulator that uses public game data to help you practice timing and animation cancelling.
*   **Comprehensive Data Reference:** A clean, local web interface displaying hero stats, ability cooldowns, and game timings pulled from official sources.
*   **Ward Map Planner:** An interactive tool to plan and study optimal ward placement based on common map control strategies.
*   **Gameplay Analysis Notes:** A curated collection of advanced tips and frame-perfect mechanics explanations.

## Why This Project?

This project is built for the European and global Dota 2 community that values **deep learning and strategic improvement**. It promotes understanding over shortcuts.

## ⚠️ Important Legal Disclaimer

This project:
*   Does NOT interact with the Dota 2 game client.
*   Does NOT provide any unfair in-game advantage.
*   Is NOT a "cheat", "hack", or "mod".

Using unauthorized software that modifies the Dota 2 client is against Valve's Terms of Service and can result in a permanent ban (VAC). This is not that.

2.  Install the required dependencies (e.g., Node.js for the web interface).
